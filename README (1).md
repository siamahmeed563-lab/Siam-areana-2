## 📁 ফাইল তালিকা

### 1️⃣ `index_updated.html` ⭐ (সবচেয়ে ভালো)
- **কী**: আপনার সম্পূর্ণ HTML ফাইল যেখানে IPTV সার্ভার যোগ করা আছে
- **কাজ**: সরাসরি ব্যবহার করুন - সবকিছু এ-তে আছে
- **বোনাস**: সব অরিজিনাল ফিচার + 12টি IPTV সার্ভার

### 2️⃣ `maruf_siam_updated.html`
- **কী**: শুধু IPTV কোড (standalone)
- **কাজ**: কপি করে নিজের ফাইলে paste করুন
- **ব্যবহার**: যদি নিজে integrate করতে চান

### 3️⃣ `INSTALLATION_GUIDE_BN.md`
- **কী**: বিস্তারিত ইনস্টলেশন গাইড
- **ভাষা**: বাংলা
- **কোড উদাহরণ সহ**

## 🚀 দ্রুত শুরু করুন

### অপশন A: সরাসরি ব্যবহার (সহজ) ✅
```
1. index_updated.html ডাউনলোড করুন
2. ব্রাউজারে খুলুন
3. "IPTV সার্ভার" বাটন ক্লিক করুন
4. যেকোনো সার্ভার নির্বাচন করুন
5. চ্যানেল খেলুন!
```

### অপশন B: নিজের ফাইলে integrate করুন (উন্নত)
```
1. maruf_siam_updated.html খুলুন
2. JavaScript কোড কপি করুন
3. নিজের HTML-এ </body> এর আগে paste করুন
4. Header-এ বাটন যোগ করুন
5. সম্পূর্ণ করুন!
```

## 🎯 কীভাবে কাজ করে?

1. **সার্ভার বেছে নিন** → IPTV সার্ভার বাটন ক্লিক
2. **M3U লোড হয়** → সার্ভার থেকে প্লেলিস্ট ডাউনলোড
3. **চ্যানেল দেখা যায়** → সবগুলো চ্যানেল লিস্টে আসে
4. **প্লে করুন** → কোনো চ্যানেল ক্লিক করে দেখুন
